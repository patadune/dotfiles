Dependencies
============

Those dependencies are available on Archlinux repos or in AUR.

For i3 :

* network-manager-applet
* dunst
* scrot
* compton
* rofi
* j4-dmenu-desktop
* xterm
* imagemagick

For mpv :

* libva-intel-driver

For bash :

* bash-completion
* pkgfile
* youtube-dl (for play/mp3 bash functions)
